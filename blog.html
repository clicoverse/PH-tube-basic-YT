Discuss the scope of var, let, and const

In JavaScript, there are three different ways to declare variables. These variables are used to define a variable name that store data or value.
'var' variable is a function scoped as well as global scoped variable. 'var' variables can be re-decleared and updated within their scope. For this reason it causes variable collisions. After ES6, for these reason 'var' variable is stoped using.
'let' variable is a block scoped variable, means their scope is limited to the block; like function, loop, conditions etc. this variables can be updated but not reassign within the same scope.
'const'varibale also a blcok scoped variable, it can't be reassign or changed after assignment. Thus this vairable is used for those values that are not meant to be modified.
